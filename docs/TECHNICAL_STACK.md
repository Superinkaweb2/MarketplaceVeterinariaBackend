# Stack Tecnológico 💻

El backend de **Huella360** utiliza un conjunto de tecnologías modernas y robustas para garantizar el rendimiento, la seguridad y la escalabilidad.

## 🛠️ Core

- **Java 17**: Lenguaje de programación principal (LTS).
- **Spring Boot 3.5.10**: Framework base para la creación de microservicios y aplicaciones web.
- **Maven**: Gestor de dependencias y automatización de construcción.

## 🗄️ Persistencia de Datos

- **PostgreSQL**: Base de datos relacional para almacenamiento persistente.
- **Spring Data JPA / Hibernate**: Abstracción de base de datos y ORM.
- **Flyway**: Herramienta de migración de base de datos para control de versiones del esquema.
- **Redis**: Almacenamiento en caché y buffer para datos de alta frecuencia.

## 🔐 Seguridad y Autenticación

- **Spring Security**: Marco de seguridad para autenticación y autorización.
- **JSON Web Token (jjwt)**: Estándar para la creación de tokens de acceso seguros.
- **BCrypt**: Algoritmo de hashing para el almacenamiento seguro de contraseñas.

## 🔌 Integraciones de Terceros

- **Cloudinary SDK**: Gestión de imágenes y archivos multimedia en la nube.
- **Mercado Pago SDK**: Procesamiento de pagos electrónicos y suscripciones.
- **Resend Java SDK**: Envío de correos electrónicos transaccionales y marketing.
- **API Perú**: Integración para validación de documentos de identidad (RUC/DNI).

## 🚀 Utilidades y Productividad

- **Lombok**: Librería para reducir el código repetitivo (Boilerplate) mediante anotaciones.
- **MapStruct**: Generador de mappers de tipos seguros para la conversión entre DTOs y Entidades.
- **Thymeleaf**: Motor de plantillas para la generación dinámica de correos electrónicos HTML.
- **Spring Validation**: Validación de datos de entrada basada en anotaciones de JSR-303.

## 📡 Comunicación

- **Spring WebSocket**: Soporte para comunicación bidireccional en tiempo real entre cliente y servidor.
- **Spring Async**: Ejecución de tareas en segundo plano para procesos no bloqueantes.
